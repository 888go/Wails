# # Vue 3 + Vite

此模板旨在帮助您开始在 Vite 中使用 Vue 3 进行开发。该模板采用了 Vue 3 的 `<script setup>` SFC（单文件组件），
要了解更多相关信息，请查阅 [script setup 文档](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup)。
